# Support

PortAtlas has no released application or support commitment yet. This repository currently supports review of its documented product and architecture foundation.

When repository issue tracking is enabled:

- Use a bug report for reproducible defects and include redacted verification evidence.
- Use a feature request for product proposals and cite affected requirements or roadmap items.
- Use the private process in [SECURITY.md](SECURITY.md) for vulnerabilities or sensitive material.

Community help is provided on a best-effort basis without a response-time guarantee. Never attach environment files, authentication tokens, proprietary source, raw process arguments, or unredacted diagnostic bundles.
